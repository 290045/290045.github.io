# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

## [1.0.0] - 2026-07-04

### Fixed
- Fixed truncated custom favicon input field in settings
- Added missing `handleLinkClick()` function that handles link navigation for all buttons
- Fixed JavaScript scope issues with panic button functions
- Reorganized script structure for better function accessibility

### Added
- Tab-based navigation system (Homepage, Games, Proxies, Tools, Settings, Credits)
- Tab cloaking feature with preset options (Google Drive, Google Classroom, Canvas)
- Custom cloak functionality for custom title and favicon
- Emergency panic button with hotkey support
- GitHub commit date display (auto-fetches last update time)
- LocalStorage persistence for cloak settings and panic key

### Features
- Games section with 10+ game links
- Proxies section with 30+ proxy links
- Tools section with educational tools
- Settings tab for customization
- Credits tab with project attribution
- Responsive grid layout for buttons
- Smooth tab switching animations
- Dark theme UI
